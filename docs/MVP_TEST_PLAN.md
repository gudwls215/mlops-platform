# MVP 테스트 유저 모집 및 관리 계획

## 1. 테스트 유저 모집 전략

### 목표
- **인원**: 20명 (최소 15명 확보)
- **기간**: 3주
- **타겟**: 50대 이상 장년층 구직자

### 모집 채널

#### 온라인 채널
1. **장년층 커뮤니티**
   - 네이버 카페: 50+, 시니어 취업, 중장년 일자리
   - 다음 카페: 재취업 준비, 실버취업
   - 페이스북 그룹: 5060세대, 중장년 취업

2. **온라인 광고**
   - 네이버 파워링크: "장년층 이력서", "50대 취업"
   - 구글 애즈: 관련 키워드 타겟팅
   - 페이스북/인스타그램: 50대+ 타겟 광고

3. **파트너십**
   - 워크넷 협력
   - 고령자 인재은행
   - 지역 일자리센터

#### 오프라인 채널
1. **커뮤니티 센터**
   - 주민센터 게시판
   - 노인복지관 공지
   - 도서관 안내

2. **직접 모집**
   - 지인 소개
   - 눈덩이 표집 (Snowball Sampling)

### 모집 메시지 예시

```
[모집] AI 이력서 작성 서비스 테스터 모집 (사례비 지급)

📢 50대 이상 구직자분들을 위한 AI 이력서 작성 서비스를 개발했습니다!

🎯 테스트 참여자 혜택
- 소요시간: 30분~1시간
- 사례비: 스타벅스 기프티콘 1만원
- 정식 출시 시 프리미엄 기능 1년 무료

✨ 이런 분들께 추천
- 이력서 작성이 어려우신 분
- AI 서비스에 관심 있으신 분
- 취업/이직 준비 중이신 분

📅 기간: 10/24 ~ 11/15
📱 참여: 온라인 (집에서 편하게)

관심 있으신 분은 댓글 또는 쪽지 주세요!
```

### 상태 추적

#### 참여 단계
1. **초대** - 초대 이메일 발송
2. **확인** - 참여 의사 확인
3. **진행중** - 테스트 진행 중
4. **완료** - 피드백 제출 완료
5. **보상** - 사례비 발송 완료

#### 피드백 수집 현황
- 총 피드백 수
- 평균 별점
- 주요 이슈/개선사항
- 긍정적 의견
- 부정적 의견

## 4. 피드백 분석 프레임워크

1. **사용량 지표**
   - 총 접속 수
   - 페이지별 방문 수
   - 평균 사용 시간
   - 기능별 사용 빈도

2. **성능 지표**
   - 평균 응답 시간
   - 에러 발생 빈도
   - API 성공률

3. **만족도 지표**
   - 평균 별점
   - NPS (Net Promoter Score)
   - 기능별 만족도

### 정성적 분석
1. **카테고리별 분류**
   - 기능성
   - 사용성
   - 성능
   - 디자인
   - 기타

2. **긍정/부정 의견 분류**
   - 칭찬/요청사항
   - 불편사항/개선사항
   - 버그/에러

3. **우선순위 매트릭스**
   ```
   | 높은 영향 | 높은 영향 |
   | 낮은 빈도 | 높은 빈도 |

   | 낮은 영향 | 낮은 영향 |
   | 낮은 빈도 | 높은 빈도 |
   ```

## 5. 리스크 관리

### 잠재적 리스크

1. **모집 부족**
   - 리스크: 15명 미만 참여
   - 대응: 추가 채널 확보, 사례비 증액

2. **피드백 부족**
   - 리스크: 피드백 미제출
   - 대응: 리마인더 발송, 개별 연락

3. **서버 장애**
   - 리스크: 테스트 중 서버 다운
   - 대응: 24시간 모니터링, 백업 서버

4. **보안 이슈**
   - 리스크: 개인정보 유출
   - 대응: 암호화, 접근 제어, 로그 관리

5. **사용성 문제**
   - 리스크: 장년층이 사용하기 어려움
   - 대응: 실시간 지원, 튜토리얼 제공
