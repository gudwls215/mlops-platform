# 🎤 음성 이력서 생성 시연 스크립트

시니어가 자연스럽게 자신의 경험을 말하는 형식으로 작성된 스크립트입니다.
각 스크립트는 이력서의 주요 항목(기본정보, 경력정보, 학력정보, 기술스택/자격증, 자기소개)이 채워질 수 있도록 구성되어 있습니다.

```

학력은 한양대학교 산업공학과를 1991년에 졸업했고요, 학사 학위입니다.

자격증은 품질경영기사, 품질관리기술사 자격을 보유하고 있고요,
```

## 💡 시연 팁

1. **자연스럽게 읽기**: 스크립트를 그대로 읽기보다 대화하듯 자연스럽게 말하세요.

2. **적절한 속도**: 너무 빠르지 않게, STT가 잘 인식할 수 있도록 또박또박 말하세요.

3. **핵심 정보 강조**: 이름, 연락처, 회사명, 직위 등 핵심 정보는 조금 천천히 말하세요.

4. **숫자 주의**: 전화번호나 연도는 특히 명확하게 발음하세요.
   - "공일공 일이삼사 오육칠팔" 또는 "010-1234-5678"

5. **조용한 환경**: 주변 소음이 적은 환경에서 녹음하세요.
